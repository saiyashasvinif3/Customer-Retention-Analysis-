# Customer-Retention-Analysis-
This is a web-based application designed to analyze customer retention using real telecom data.It helps in gaining meaningful insights through visual analysis to understand customer churn patterns.The project uses Python along with Matplotlib and Seaborn to suggest better business outcomes.

# Project Overview
This project analyzes customer churn for a telecom company to understand why customers leave. Using the Telco dataset, we explore patterns in customer behavior, visualize important trends, and identify factors that influence churn. The goal is to help the company retain customers and improve services.

## Features
- Upload and analyze customer churn dataset easily  
- Clean and preprocess data for accurate analysis  
- Visualize customer retention and churn patterns  
- Generate insights using charts and plots
- Save visualizations as images for reports and presentations  
- Identify key factors affecting customer retention  
- Simple and user-friendly workflow
  
## Technologies Used
- **Python** – Core programming language used for data analysis  
- **Pandas & NumPy** – Used for  data loading, cleaning, and processing  
- **Matplotlib** – Used for creating and saving visualizations  
- **Seaborn** – Used for advanced statistical plots  
- **Visual Studio Code** – Used for interactive analysis and code implemtation
